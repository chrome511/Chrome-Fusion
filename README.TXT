2. Deutsch update

############################################
Update von Chrome-Fusion 1.0.0 to 2.00.00
############################################

Enthält nur die aktualisierten Dateien.
Mit diesem Paket kann eine originale Version von Chrome-Fusion v1.0.0 aktualisiert werden, um unter neueren Chrome-Versionen wieder fehlerfrei verwendet zu werden.
Hierfür einfach alle enthaltenen Dateien in den Ordner der Chrome-Fusion Homepage hineinkopieren, dabei unbedingt die Ordnerstruktur einhalten und vorhandene Dateien überschreiben lassen.
Die vorhandene Datei "config.php" muss auf dem Webspace um folgende Zeile ergänzt werden:

$db_driver = "mysqli";

Dafür am besten vor der letzten Zeile mit dem ?> eine Lerzeile einfügen und dort hineinkopieren.
Möchte man PDO Anbindung verwenden, einfach statt "mysqli" das "pdo" eintragen:

$db_driver = "pdo";

(Lässt man diesen Schritt einfach komplett weg, wird autom. per MySQLi mit der Datenbank kommuniziert.)

----------

Neben den nötigen Updates bringt dieses Pack folgende neue Features:
- TinyMCE v4
- Wählbare Hintergrundfarbe für den TinyMCE-Editor (Einstellungen -> Sonstiges, beim Original nicht wählbar)

Weitere Features habe ich bewusst weg gelassen, um möglichst nahe am Original zu bleiben und vorerst hauptsächlich als Fehlerkorrektur zu dienen.

Wie immer gilt:
- Backup vorher erstellen!
- Keine Gewährleistung!

----------
Sie können von keiner kleineren Version aktualisieren.
Dies ist nur für Version 1.0.0
----------

- Laden Sie alle Dateien aus dem \update v1.0.0\files\ Ordner hoch. (Ordnerstruktur beachten!)

- Führen sie das Upgrade über das Administrator Menü aus.

- folge den Anweisungen.




Irgendwelche Fehler?
Besuche: chrome-fusion.de for support ;)

4. English update

############################################
Update from Chrome-Fusion 1.0.0 to 2.00.00
############################################

Contains only the updated files.
With this package, an original version of Chrome Fusion v1.0.0 can be updated to be used correctly on newer versions of Chrome.
For this simply copy all contained files into the folder of the Chrome-Fusion homepage, while keeping the folder structure and overwrite existing files.
The existing file "config.php" has to be added to the webspace with the following line:

$db_driver = "mysqli";

This is best before the last line with the?> Insert a line and copy it into it.
If you want to use PDO connection, simply enter "pdo" instead of "mysqli":

$db_driver = "pdo";

(If you just leave this step completely, it automatically communicates with the database via MySQLi.)

----------

In addition to the necessary updates, this pack brings the following new features:
- TinyMCE v4
- Selectable background color for the TinyMCE editor (Settings -> Other, not selectable for the original)

I deliberately omitted other features in order to stay as close to the original as possible and for the time being serve mainly as an error correction.

As always:
- Create backup in advance!
- No warranty!
